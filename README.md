## 💻 Como usar
- ***Clone o projeto:*** <br>
``` git clone https://github.com/lmiguelm/imc-app.git```

- ***Entre na pasta do projeto:*** <br>
 ```cd imc-app```
 
- ***Baixe as dependências:*** <br>
``` yarn || npm install```

- ***Rodar o app:*** <br>
```yarn start || npm start```
